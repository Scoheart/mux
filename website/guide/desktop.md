# 桌面 App 指南

桌面 App 是 MUX 的可视化前端（macOS，Tauri + React），适合用鼠标浏览目录、勾选安装、编辑条目。它和命令行共享 `~/.mux/`，两端实时同步。

> 还没装？看 [安装](/guide/install#桌面-app-macos)。

## 界面导航

App 顶部导航分三块：

- **Registry（目录）** —— 聚合所有已启用来源的 MCP 列表。
- **Sources（来源）** —— 管理订阅 / 导入 / 手动 / 探索来源，带一个来源侧边栏。
- **每个 Agent** —— 逐个 agent 查看/管理它当前装了哪些 MCP。

加上一个全屏的**编辑器**页，用来编辑或粘贴目录条目。

## 浏览目录

在 Registry 页：

- **搜索**：按名字过滤。
- **过滤**：按来源、传输方式（stdio / http / sse）过滤。
- 每个卡片显示：**来源徽章**（手动 / 探索 / 远程 / 本地）、传输方式、被哪些 agent 使用；若设置了仓库 / 主页，还会显示其**源仓库链接**。

## 安装到 agent

1. 在目录里选中一个 MCP，打开它的详情。
2. 勾选要安装到的 agent（可多选）。
3. 确认写入。MUX 会：
   - **先备份**目标 agent 的配置文件到 `~/.mux/backups/`；
   - 只**改写这一个 server 条目**，保留同文件里其它 server 的原始字节；
   - 用该 agent 各自的格式（JSON / TOML）和键名写入。

## 开 / 关 / 删除

对每个 agent 的每个已装 MCP：

- **停用**：从 agent 文件移除，但记住配置——随时可开回来。
- **启用**：把停用的写回去。
- **删除**：从该 agent 卸载。

在目录里对 **manual / 探索** 条目，还有一个**删除**按钮，会把它从目录彻底移除并从所有 agent 卸载（详见 [核心概念 · Forget](/guide/concepts#删除目录条目-forget)）。

## 编辑与粘贴

- **编辑**：可视化编辑一个 MCP 的配置字段。保存后，编辑会**自动传播**到干净安装了它的 agent（见 [编辑传播](/guide/concepts#编辑传播-edit-propagation)）。
- **粘贴**：直接粘一段 `{"mcpServers": {…}}`（或 TOML）配置块，MUX 会识别出其中的 server 并加入手动来源。
- **重新同步**：编辑器里的「重新同步」按钮，把当前配置显式推给所有已安装该 MCP 的 agent；被定制过的会被跳过并报告，可选择强制覆盖。

## 来源管理

在 Sources 页：

- **订阅**一个远程 URL；
- **导入**一个本地文件；
- 一键添加**官方精选合集**；
- **刷新**（重新拉取上游 / 重读原文件）；
- **启停**、**删除**来源。

## Agent 管理

- **新增自定义 agent**：给它起名、指定配置文件路径和格式。
- **编辑** agent 的配置文件路径（路径以 `~/…` 可移植方式存储）。
- **启停** agent。

支持的 20 个内置 agent 见 [支持的 Agent](/guide/agents)。

## 安全性

- **原子写入** + 每次触碰 agent 配置前的**时间戳备份**。
- **按条目改写**，绝不整体重写 `mcpServers` 映射——保护同文件里的其它 server。

命令行做的是同样的事，只是在终端里 → [命令行 / TUI](/guide/cli)。
